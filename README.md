## Sobre o Hackathon

Este projeto foi desenvolvido como parte do **Hackathon Fullstack - Trilha PU12**, com foco na criação de um **frontend de e-commerce moderno, funcional e responsivo**.

O hackathon é um evento colaborativo e competitivo com duração limitada, com o objetivo de:

* Estimular criatividade e inovação
* Resolver desafios reais em curto prazo
* Aplicar conhecimentos na prática
* Promover trabalho em equipe e networking

---

## Objetivo do Projeto

Desenvolver uma aplicação frontend de e-commerce aplicando conceitos de:

* Componentização
* Responsividade
* Usabilidade
* Organização visual
* Experiência do usuário

A proposta é simular uma experiência real de compra, utilizando dados mockados, sem necessidade de um backend robusto.

---

## Tema do Desafio

Criar um **frontend de e-commerce** com:

* Interface clara e organizada
* Experiência de compra intuitiva
* Visual moderno e atrativo

---

## Escopo Técnico

O projeto contempla:

* Estruturação de páginas
* Navegação entre telas
* Componentes reutilizáveis
* Integração com dados mockados
* Simulação de fluxo de compra

---

## MVP (Mínimo Produto Viável)

Funcionalidades obrigatórias:

* Página inicial (banner + produtos + categorias)
* Listagem de produtos
* Página de detalhes do produto
* Carrinho (controle de quantidade + total)
* Checkout simplificado
* Identidade visual consistente

---

## Funcionalidades Extras

Diferenciais implementáveis:

* Busca de produtos
* Filtro por categorias
* Favoritos
* Perfil do usuário
* Meus pedidos
* Banners promocionais
* Responsividade mobile avançada
* Simulação de login

## Como Executar

```bash
# Clone o repositório
git clone https://github.com/skynetsites/hackastore.git

# Instale dependências
npm install

# Execute backend (mock)
npm run start:prod

# Execute frontend
cd frontend
npm install
npm run dev
```
